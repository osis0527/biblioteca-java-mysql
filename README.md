# 📚 Ejercicio 20 – Conexión Java a Base de Datos

## 📌 Descripción
Aplicación Java que se conecta a MySQL para gestionar
el catálogo de una biblioteca. Implementa el patrón DAO
para separar la lógica de acceso a datos.

## 🎯 Conceptos practicados
- Conexión Java a MySQL con JDBC
- Patrón DAO (Data Access Object)
- PreparedStatement para consultas seguras
- ResultSet para leer resultados
- CRUD completo desde Java

## 🚀 Cómo ejecutar
1. Instala MySQL y crea la base de datos biblioteca_db
2. Configura usuario y contraseña en ConexionDB.java
3. Agrega mysql-connector-j al proyecto
4. Compila y ejecuta Main.java

## 🛠️ Tecnologías
- Java 17
- MySQL 8.0
- MySQL Connector/J 9.7.0
